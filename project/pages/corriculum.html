Full Stack Web Development Curriculum

Learn HTML, CSS, JavaScript, Backend, APIs, AI, Databases, and Full Stack Development through real-world projects.

🏗️

Phase 1 - HTML

Learn the structure of websites using only HTML and create multi-page websites.

Skills Learned

  • HTML tags & elements
  • Forms & tables
  • Multi-page websites
  • Website structure
Project: Student Profile & Course Portal
🎨

Phase 2 - CSS

Style websites with colors, layouts, spacing, shadows, and responsive design.

Skills Learned

  • Colors & typography
  • Flexbox & layouts
  • Spacing & alignment
  • Responsive design
Project: Styled Student Portal Website
⚡

Phase 3 - JavaScript

Make websites interactive using JavaScript with buttons, logic, and dynamic features.

Skills Learned

  • Variables & functions
  • DOM manipulation
  • Events & conditions
  • User interactions
Projects: Todo App, Calculator, Quiz App
🤖

Phase 4 - Modern JS & AI

Learn ES6 JavaScript and build AI-powered applications using Gemini API.

Skills Learned

  • ES6 concepts
  • API integration
  • Fetch & async JS
  • AI content generation
Project: AI Content Generator
🚀

Phase 5 - Backend Development

Build backend servers using Node.js, Express.js, and Gemini AI integration.

Skills Learned

  • Node.js basics
  • Express.js server
  • REST APIs
  • Frontend & backend connection
Project: Full Stack AI Web App
💾

Phase 6 - MongoDB & Mongoose

Learn databases, data storage, and backend data management systems.

Skills Learned

  • MongoDB database
  • Mongoose schemas
  • CRUD operations
  • Store & manage data
Project: Student Feedback Manager
